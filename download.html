System Requirements

  • Operating System: Windows 10/11 (64-bit)
  • Game: Grand Theft Auto V (latest version)
  • .NET Framework: 4.8 or higher
  • Dependencies: Visual C++ Redistributable
  • Disk Space: ~50 MB for framework files

Quick Installation Guide

  1. Download the Framework

    Click the download button above to get the latest version of Script Hook V .NET

  2. Extract Files

    Extract the contents of the downloaded archive to a temporary location

  3. Install to GTA V Directory

    Copy all files to your Grand Theft Auto V main directory (where GTA5.exe is located)

  4. Create Scripts Folder

    Create a "scripts" folder in your GTA V directory if it doesn't exist

  5. Add Your Mods

    Place your .dll mod files in the "scripts" folder

  6. Launch the Game

    Start GTA V and your mods should load automatically
